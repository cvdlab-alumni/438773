Work Description

Costruzione dell'appartamento

Nel seguente progetto è stato rappresentato una appartamento utilizzando la libreria sysMl per la costruzione di stanze porte e finestre. La costruzione è stata fatta in maniera incrementale andando a costruire una stanza per volta, stessa cosa per le finestre e per le porte. Successivamente e' stato assemblato un condominio sempre usando sysml posizionando al piano terra una serie di superficie curve.
